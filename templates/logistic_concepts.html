Understanding Logistic Regression applied to E-Commerce Purchase Prediction.

What is Logistic Regression?

Logistic Regression is a supervised ML algorithm used for binary classification — it predicts whether something belongs to one of two categories (0 or 1, Yes or No, True or False). Unlike Linear Regression which outputs a continuous number, Logistic Regression outputs a probability between 0 and 1.

In this model: We predict whether an e-commerce purchase is Premium (Final Price ≥ 300 Rs.) or Standard (Final Price < 300 Rs.) based on the product price, discount, category and payment method.

The Sigmoid Function

The key function that converts any number into a probability between 0 and 1:

σ(z) = 1 / (1 + e⁻ᶻ)

Where z = b₀ + b₁x₁ + b₂x₂ + ... (the linear combination of inputs).

  • If σ(z) ≥ 0.5 → predict Premium (1)
  • If σ(z) < 0.5 → predict Standard (0)

How the Model Was Trained
1
Dataset
3,660 real e-commerce transactions with price, discount, category and payment method.
2
Encoding + Scaling
Categories and payment methods were label-encoded. Numeric features were standardized with StandardScaler.
3
Train / Test Split
80% training (2,928 rows), 20% testing (732 rows). Random state = 42.
4
Result
The model achieved 99.18% accuracy on the test set, correctly classifying premium and standard purchases.
